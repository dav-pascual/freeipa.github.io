Overview¶

The goal of the documentation provided here is:

  • To explain what FreeIPA is, the issues it addresses, and how it fits into an organization’s IT infrastructure.

  • To help an administrator to quickly install, set up, and administer one or more servers for centralized authentication and identity management.

  • To help users perform everyday tasks related to freeIPA.

  • To help identify common problems and how to solve them.
