FreeIPA team welcome help in all documentation related areas, especially with adding articles, HOWTOs or just updating information on this community wiki.

Updating wiki¶

This project site is running on famous mediawiki engine, so updating it is very easy. You just need to log in with your Fedora project account and start contributing!

What to do¶

You can update or extend current articles referenced from Documentation or add new content you would like to share with the FreeIPA world. You can also check FreeIPA Trac for the list of known bugs or missing enhancements to this wiki and help us!

Contributing downstream guides¶

FreeIPA project is documented in several downstream User Guides which contain enterprise level documentation. Downstream guide maintainers are happy for any contribution, from proposed content to even just a bug report.

  • Direct links for bug reporting can be found next to links to downstream guides.
