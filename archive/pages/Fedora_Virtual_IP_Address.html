Virtual Address¶

Select an existing network interface (e.g. eth0). Create a new network interface as its child:

% cd /etc/sysconfig/network-scripts
% cp ifcfg-eth0 ifcfg-eth0:0

Edit ifcfg-eth0:0:

DEVICE=eth0:0
IPADDR=<new IP address>
# BOOTPROTO=dhcp

Restart networking service:

% service network restart
